Section "Création de contenu" à construire. Deux modes :

**Mode 1 — Brainstorm** : Le client veut des idées et des key points, pas un script complet. Il brainstorme à voix haute ou par écrit. L'IA propose des angles basés sur le Brand OS + Journal. Résultat : des directions, pas un texte fini.

**Mode 2 — Script** : Le client veut un script complet (YouTube, Instagram, etc.). L'IA utilise le Brand OS + Journal + Calendar pour écrire un script aligné avec sa voix, son audience, son message.

**Fonctionnalités requises :**
- Note vocale (Web Speech API) intégrée dans les deux modes
- L'IA pioche dans le Journal (notes du client) pour proposer des angles authentiques
- L'IA croise avec le Brand OS (audience, problèmes, voix, valeurs) pour rester aligné
- L'IA vérifie l'équilibre TOFU/MOFU/BOFU et alerte si déséquilibre
- Transcription vocale pour les clients qui préfèrent parler que écrire

**Why:** Jonathan veut que la création de contenu soit le cœur du SaaS. Tout le Brand OS et le Journal alimentent cette section.

**How to apply:** Quand on construit cette section, s'assurer que les deux modes sont clairs dès l'entrée. Le client choisit "Brainstorm" ou "Script" et le flow s'adapte.
