# Brvndlab Memory

Backup GitHub des règles mémoire et décisions produit du projet Brvndlab.

Ce dossier contient **toutes les règles, décisions, préférences et contextes** que Claude utilise comme source de vérité à travers les sessions. Il sert de sauvegarde hors-ligne au cas où.

## Structure

- **`MEMORY.md`** : index principal, liste toutes les règles avec un lien vers leur fichier
- **`VALIDATED.md`** : liste de tous les mockups validés par Jonathan, avec leur statut
- **`feedback_*.md`** : règles comportementales (comment Claude doit agir)
- **`project_*.md`** : décisions stratégiques et architecturales
- **`reference_*.md`** : pointeurs vers des ressources externes
- **`user_jonathan.md`** : profil de Jonathan

## Mise à jour

Ce dossier est mis à jour manuellement par Claude à chaque étape majeure (fin de Chantier A, décisions de fondation technique, etc.). Il reflète l'état de la mémoire au moment de la dernière sync.
