Ne jamais utiliser le caractère em-dash dans aucune communication avec Jonathan. Ni dans le chat, ni dans les mockups, ni dans le copy de l'app Brvndlab. Aucune exception.

**Why:** Préférence personnelle forte de Jonathan. Il l'a explicitée le 2026-04-07 après avoir vu plusieurs mockups où j'avais utilisé l'em-dash dans les titres et les phrases. Sa formulation : "ça je te dis, je ne veux jamais dans ma communication".

**How to apply:**
- Remplacer les em-dashes par : virgules, points, deux-points, parenthèses, ou retour à la ligne
- Vérifier systématiquement avant d'envoyer un message ou de générer un mockup
- Aussi vérifier dans les contextes envoyés à Gemini (sinon Gemini les inclut dans le HTML généré)
- Si je vois un em-dash dans un fichier que je modifie, le remplacer même si ce n'est pas l'objet de la modification
